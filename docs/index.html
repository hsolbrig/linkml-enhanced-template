Person schema

Classes

  • Thing - The most generic type of item.
    • Person - A person (alive, dead, undead, or fictional).

Mixins

Slots

  • additionalName - An additional name for a person, can be used as a middle name.
  • gender - Person gender
  • givenName - Given name. In the U.S., the first name of a Person.
  • identifier - The identifier property represents any kind of identifier for any kind of Thing, such as ISBNs, GTIN codes,
  • name - The name of the item.
  • url - URL of the item
